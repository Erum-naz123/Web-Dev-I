# Chocolate Chip Cookies – Semantic HTML Exercise

This project is a simple webpage created using **semantic HTML elements**.

## Description
The page displays a chocolate chip cookie recipe, including:
- Ingredients list
- Step-by-step instructions

## Features
- Uses semantic tags such as `header`, `main`, `section`, and `footer`
- Proper use of headings (`h1`, `h2`)
- Includes both unordered and ordered lists
- Clean and readable HTML structure

## Purpose
This exercise demonstrates understanding of:
- Basic HTML structure
- Semantic markup
- Code formatting and organisation
