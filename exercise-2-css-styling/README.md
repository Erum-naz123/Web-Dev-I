# Set Exercise 2: CSS Styling

## Overview
This assement task recreates a webpage using semantic HTML and an external CSS stylesheet. The content is based on the provided *Our Story* text and focuses on applying styling techniques such as typography, layout, and colors.

## Features
- Semantic HTML structure (`header`, `main`, `section`, `footer`)
- External CSS stylesheet for styling
- Custom fonts, colors, and spacing
- Responsive layout with constrained content width
- Clean and well-indented code
